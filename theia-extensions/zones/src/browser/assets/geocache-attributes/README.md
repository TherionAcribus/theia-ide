# assets/geocache-attributes

Ce répertoire contient les images PNG sources des icônes d'attributs de géocaches.

**NE PAS SUPPRIMER** - Ce répertoire est nécessaire pour :
- Ajouter de nouvelles icônes
- Modifier des icônes existantes
- Régénérer le fichier de données base64

Voir GESTION_IMAGES_ATTRIBUTS.md pour la documentation complète.
